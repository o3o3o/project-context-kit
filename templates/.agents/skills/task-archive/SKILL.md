# task-archive Skill

## Purpose
Move the active GCC memory tree from `active/` to `archive/` once the mission is complete.

## Instructions

1. **Verify `verification.md`**: Ensure the final verification report is complete and signed off.
2. **Move to Archive**: `.ai-governance/docs/task/archive/YYYY-MM-DD-[task-name]/`.
3. **Final Update**: Update the archived `summary.md` one last time to status "COMPLETED".
4. **Clear Active**: Empty `.ai-governance/docs/task/active/`.
5. **Report**: *"Mission archived. Ready for next task."*
