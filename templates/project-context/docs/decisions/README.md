# Decisions

This directory stores lightweight long-lived design decisions.

## When To Write Here
- Write here only for project-level decisions that should outlive the current task.
- Do not use this directory for session notes, temporary reasoning, or active task handoff.

## Minimal Format
Each decision file should stay short and usually include:
- `Status`
- `Context`
- `Decision`
- `Consequences`
- `When To Revisit`

## Working Rule
If a conclusion is still important after the current task is archived, record it here and reference it from the active task summary or project context.
